# 🚀 SuperApp на Python

Многофункциональное приложение с графическим интерфейсом для повседневных задач.

---

## 📋 Функции

### ✅ Реализовано (v1.0)

1. **Курсы валют ЦБ РФ**
   - Отображение текущих курсов всех валют
   - Интерактивный график истории курса
   - Обновление в реальном времени
   - Кэширование данных

### 🚧 В разработке (v2.0)

2. **Прогноз погоды** (заглушка)
3. **Заметки** (заглушка)
4. **Калькулятор** (заглушка)
5. **Список дел** (заглушка)

---

## 🎨 Выбор UI-фреймворка

### PyQt5

**Выбран по следующим причинам:**

| Критерий | PyQt5 |
|----------|-------|
| Кроссплатформенность | ✅ Windows, macOS, Linux |
| Визуализация графиков | ✅ QChart встроен |
| Профессиональный вид | ✅ Нативные виджеты |
| Документация | ✅ Обширная |
| Сообщество | ✅ Большое |
| Лицензия | ✅ GPL для открытых проектов |

**Альтернативы, которые рассматривались:**

- **Tkinter** — слишком простой, нет графиков
- **Kivy** — ориентирован на мобильные приложения
- **PySide6** — аналог PyQt5, но менее популярен в России
- **wxPython** — устаревший

```bash
